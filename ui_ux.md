- Regular: 400
- Medium: 500
- Semibold: 600

## Headings

**H1 - Page Titles:**

- Size: 32px (2rem)
- Weight: Bold (700)
- Color: Dark Gray (#1f2937)
- Line Height: 1.2
- Usage: Main page title only

- Size: 24px (1.5rem)
- Weight: Semibold (600)
- Color: Dark Gray (#1f2937)
- Line Height: 1.3
- Usage: Major sections

**H3 - Subsection Titles:**

- Size: 18px (1.125rem)
- Weight: Semibold (600)
- Color: Dark Gray (#1f2937)
- Line Height: 1.4
- Usage: Card titles, subsections

**Body - Regular:**

- Size: 14px (0.875rem)
- Weight: Regular (400)
- Color: Dark Gray (#1f2937)
- Line Height: 1.5
- Usage: Main content, descriptions

- Size: 12px (0.75rem)
- Weight: Regular (400)
- Color: Medium Gray (#6b7280)
- Line Height: 1.4
- Usage: Metadata, timestamps, helper text
